# Contributing

This repository is proprietary and is not open source.

Virtual Research Institute is not currently accepting public code contributions, pull
requests, design submissions, program materials, research-project materials, prompts,
or other outside contributions through this repository unless there is a separate
written agreement with Virtual Research Institute LLC.

Please do not submit unsolicited pull requests or contributions containing code,
content, designs, program structures, curriculum materials, research-project ideas,
student materials, business materials, or other intellectual property.

Public availability of this repository on GitHub does not grant permission to copy,
reuse, modify, distribute, commercialize, or create derivative works from the materials
in this repository.

Security issues should not be reported through public issues or pull requests. Please
follow the instructions in `SECURITY.md`.
